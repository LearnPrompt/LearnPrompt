LearnPrompt 不是工具名词表，也不是从第一页背到最后一页的课本。它是一套面向真实项目的中文 AI 实战教程：先选择一个正在发生的任务，再沿着最短路径做出可检查的结果。

## 每篇教程怎样证明自己有用

深度教程不会只给概念和截图。你会看到四类证据：

1. **明确的读者任务**：读完以后要能做什么，而不是“了解某功能”。
2. **可重放的 Showcase**：有效场景之外还包含会被拒绝的反例。
3. **可执行验收**：命令、退出码、diff、receipt 或人工检查点。
4. **真实来源与边界**：一手资料、二手主题地图和现场实验分开标注。

`verified` 只表示文章已经通过项目内的研究、Showcase、独立评审、图片和构建门禁；它不代表工具能力永远不会变化。遇到版本、价格、权限或安全问题，仍应沿文章底部链接回到官方资料核对。

## 最短的使用方法

```text
选一个真实任务
  → 只读当前阻塞所需的 1–3 篇
  → 复制模板并改成自己的输入
  → 跑命令或人工验收
  → 把失败写回项目规则、记忆或 Skill
```

如果你只有 15 分钟，完成一篇教程里的练习；如果你有一周，按 [7 天实践路线](/start-here/ai-practice-map/#7-天实践路线) 跑完一次闭环。不要把“收藏过”当作“已经会用”。

## 使用前先守住边界

第一次让 Agent 写文件、运行命令或访问外部系统前，先读 [AI 工具配置的安全底线](/start-here/setup-safety-basics/)。涉及密钥、删除、发布、付费调用或真实用户数据时，教程不能替你做授权决定。
