## 原则

旧 Docusaurus 内容暂不删除。后续迁移时每个页面只进入三种状态：更新、过时、归档。

## 三类处理

- 更新：Prompt 基础、RAG 基础、Agent 基础、长期有效学习方法。
- 过时：旧模型对比、旧工具界面、旧新闻解读、旧 GPTs 教程。
- 归档：不可复现、只剩历史参考价值、外部依赖已失效的页面。

## URL 策略

旧 URL 默认保留或设置重定向。新版页面会提供替代入口，旧页顶部加状态提示，避免用户从搜索引擎进入后误用过时教程。

## 当前阶段

本阶段只搭建 Starlight 新站和首批教程。旧内容实际迁移将在构建稳定后逐页处理。
