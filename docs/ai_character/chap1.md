# 简介

AI虚拟数字人是基于人工智能技术创建的虚拟智能体系。他们通能够过大量的数据学习，模拟人类的思维过程和决策模式，实现各种智能应用。

你有可能用过的Siri，小爱同学和天猫精灵，他们背后的技术就是AI数字人。
