## 核心概念

内容自动化不要一开始做成“全自动发布”。先把每一步结果落盘，允许人工检查，再逐步自动化。

## 流水线

1. 抓取：保存原始数据和时间戳。
2. 清洗：统一字段，不丢弃异常。
3. 筛选：记录规则和命中原因。
4. 摘要：保留来源链接。
5. 发布：先生成草稿，再人工确认。

## 常见错误

- 把抓取失败当作“今日无内容”。
- 摘要没有来源，无法追溯。
- 没有历史快照，无法做增量榜单。

## 实战案例

ClawHub 榜单日报应保存当天 JSON 快照，再和前一日比较。缺少历史切片时，日报必须写明“不能做严格日环比”。
